This plugin allow you warn visitors when they want to go in category.
== Description ==

This plugin is very usefull if you want warn people going to one category.

It set cookie, co visitor will see alert message only once.

For example you have a magazine and in one category you have adult content. 
With this plugin you can set up warning for access to this category-and user have to confirm your message. 
  
== Installation ==

1. Unzip
2. Upload `wp warning` to the `/wp-content/plugins/` directory
3. Activate the plugin through the 'Plugins' menu in WordPress
4. Choose category with warning and set up your message.
